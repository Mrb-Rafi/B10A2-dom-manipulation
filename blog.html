What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for representing HTML, XML, and other structured documents as a tree of nodes. Each node represents a part of the document, such as an element, attribute, text node, or comment. The DOM provides a way to access and manipulate these nodes using JavaScript or other programming languages.

How do you select an element from the DOM?

To select an element in the DOM, use getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll. You can also use references to parent elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM refers to the practice of attaching event listeners to a parent element instead of directly to each individual child element. This allows you to handle events for multiple elements without having to write separate event listeners for each one.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, use setAttribute, getAttribute, and removeAttribute. To manipulate styles, use style.propertyName, classList.add, and classList.remove.
